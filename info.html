What is mediaReq?

mediaReq is a platform for individuals to add youtube videos to a queue, and have them play in order.

What are some of the technolgies used for this website?

The website is built in pure HTML, CSS, Javascript, and Jquery.

What are the jquery plugins used for the website?

The plugins used are as follows:

  • metisMenu
  • Stellarnav
  • Generic jquery accordion

Where are the plugins used?

The plugins are used in the following ways:

  • metisMenu: used to beautify the sitemap page, giving a more professional look to the page.
  • Stellarnav: used to create the header that is seen on all pages except for the landing page.
  • Generic Accordion: used to create the interactive playlist viewer on the dashboard

Where is AJAX used in the website?

AJAX is used in the following ways:

  • Getting video thumbnails in the dashboard
  • Getting video names in the dashboard
  • Getting video descriptions in the dashboard
